-----DESCRIPCION-----
Sistema de gestion de stock para una tienda.
Permite administrar un catalogo de productos de tres tipos (electronica,
ropa y alimentos), registrar clientes, realizar ventas con control de
stock, reponer mercaderia y consultar informacion del catalogo.

Los datos del catalogo se guardan en un archivo (productos.dat)

------FUNCIONALIDADES------
El sistema ofrece, a traves de un menu por consola, las siguientes opciones:
1. Agregar producto (electronica, ropa o alimento)
2. Registrar cliente
3. Vender producto 
4. Reponer stock
5. Mostrar productos disponibles
6. Mostrar alerta de productos con stock bajo
7. Mostrar catalogo ordenado por precio
8. Salir 

-----EJECUCION-----
Ejecutar la clase Main.java (ubicada en el paquete tienda), 
Seguir las opciones del menu que aparece en la consola.
